# Multi-Agent Workflow

SlopScore.ai uses **5 autonomous agents** that work in parallel:

---

## 1. Claim Verifying Agent
**What it does**: Extracts factual claims from transcript and 
verifies them against authoritative sources.

**Output**: TRUE/FALSE/COULDN'T_TELL verdict for each claim

[Learn More →]

---

## 2. Author Reputation Agent
**What it does**: Investigates video creator's credibility and 
cited sources.

**Output**: Source credibility scores + citation chain analysis

[Learn More →]

---

## 3. Visual Analysis Agent
**What it does**: Checks thumbnail and key frames for AI artifacts.

**Output**: AI generation detection + reverse image search results

[Learn More →]

---

## 4. Supervisor Agent
**What it does**: Orchestrates all agents and compiles final report.

**Output**: Overall SlopScore + Trust Report

[Learn More →]

---

## Why Multi-Agent?

- ✅ Parallel processing (faster)
- ✅ Specialized expertise (more accurate)
- ✅ Modular (easy to update individual agents)
- ✅ Transparent (users see each agent's reasoning)
